Editor

Basic part of the application is the editor. In this mode, different tools are enabled in the top toolbox. It allows the user to interact with the net in the following way:

  • Select tool: Click and drag an item to select and move it. Hold CTRL to select multiple items. Right click&drag to select a rectangle. Press delete to remove selected entities.
  • Add place: Click the scene to add place.
  • Add transition: Click the scene to add transition.
  • Add arc: Click and drag line to create an arc between place and transition. Creation of duplicite arcs is not possible.
  • Delete entity: Click entities on the scene to delete them.
  • Edit inscription: At any time depending on the item selected.
To proceed to the net simulation, press the Compile button. If an error is found, compilation output is populated with its description. Click any of the rows to focus on responsible entity/inscription.
