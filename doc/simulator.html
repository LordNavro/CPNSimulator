Simulator

Simulator allows the user to make net computations. Use the following actions to interact with the net:

  • Manually fire transition: Select binding element from the combo box and click the Fire button.
  • Move to initial marking: Populate places using the initial marking inscriptions.
  • Fire random transition: Click this action to fire a random enabled transition with a random binding element.
  • Fire multiple: Input transition count and random transition will get fired.
  • Drag items: Drag items in case the net is not readable enough.

To return to edit mode, click the Edit net button. To proceed to analysis, click Analysis -> Generate state space.

In case that any of the computation takes too long to execute or an endless loop is reached, you can manually terminate it using the Cancel button. If it was the Find binding action that you have terminated, you can recall it manually using the Find binding button. However, most of these cases are caused by errors in net design and return to Editor is suggested.
